# homebrew-tap

Homebrew tap for `shinypancake` developer tools.

## worktree-lanes

Per-worktree/lane dev+test isolation CLI ([shinypancake/worktree-lanes](https://github.com/shinypancake/worktree-lanes)).

```bash
brew install shinypancake/tap/worktree-lanes
worktree help
```

Upgrade with `brew upgrade worktree-lanes`.
